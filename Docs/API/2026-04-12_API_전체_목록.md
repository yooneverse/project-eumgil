```json
{
  "status": "C4000",
  "message": "잘못된 입력입니다."
```

## 6. 공통 Error Code

| HTTP Status | status | message | 발생 조건 |
| --- | --- | --- | --- |
| 400 | C4000 | 잘못된 입력입니다. | 요청값 검증 실패 |
| 401 | A4010 | 인증이 필요합니다. | 토큰 누락/만료 |
| 403 | A4030 | 권한이 없습니다. | 권한 없음 |
| 404 | C4040 | 리소스를 찾을 수 없습니다. | 조회 대상 없음 |
| 409 | C4090 | 이미 처리된 요청입니다. | 중복/상태 충돌 |
| 500 | I5000 | 서버 내부 오류가 발생했습니다. | 서버 예외 |
| 502 | E5020 | 외부 API 호출에 실패했습니다. | 카카오 API 등 외부 API 실패 |
