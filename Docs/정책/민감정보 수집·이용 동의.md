# 민감정보 수집·이용 동의

운영자는 맞춤형 접근성 안내를 제공하기 위하여 아래와 같이 민감정보를 수집·이용합니다. 민감정보는 일반 개인정보와 구분하여 별도로 동의를 받으며, 이용자는 본 동의를 거부할 수 있습니다.

## 1. 수집·이용 항목

운영자는 다음의 민감정보를 수집·이용할 수 있습니다.

1. 이용자가 선택한 사용자 유형
2. 장애 유형 또는 장애 관련 선택 정보
3. 이동 방식
4. 보조기기 사용 여부
5. 보행 또는 이동 특성
6. 접근성 선호 설정
7. 이용자가 직접 입력하거나 선택한 접근성 관련 요청사항

## 2. 수집·이용 목적

수집한 민감정보는 다음 목적을 위해 사용됩니다.

1. 이용자가 선택한 조건에 따른 맞춤형 경로 안내
2. 접근성 정보 필터링
3. 장소 및 경로 안내 화면 개인화
4. 이동 방식과 접근성 선호도를 반영한 장소 정보 제공
5. 오류 확인 및 기능 개선

단, 통계 작성, 품질 개선, 기능 개선 목적으로 활용하는 경우에는 특정 개인을 알아볼 수 없도록 처리한 정보만 사용합니다.

## 3. 보유 및 이용 기간

1. 민감정보는 서비스 이용 기간 동안 보유·이용합니다.
2. 이용자가 해당 설정을 삭제하거나 동의를 철회하거나 서비스에서 탈퇴하는 경우 지체 없이 파기합니다.
3. 백업, 장애 대응, 분쟁 대응 등 정당한 사유로 일시 보관이 필요한 경우에는 해당 목적에 필요한 최소 범위에서 분리 보관한 뒤 파기합니다.
4. 법령상 보관 의무가 있는 경우에는 해당 법령에서 정한 기간 동안 보관할 수 있습니다.

## 4. 동의 거부 권리 및 불이익

1. 이용자는 민감정보 수집·이용에 대한 동의를 거부할 수 있습니다.
2. 동의하지 않거나 동의를 철회하는 경우 맞춤형 접근성 경로 안내, 사용자 유형 기반 추천, 접근성 필터 기능이 제한될 수 있습니다.
3. 다만, 기본 지도, 장소 검색, 수동 경로 조회 등 민감정보가 필요하지 않은 기능은 계속 이용할 수 있습니다.

## 5. 민감정보 보호조치

운영자는 민감정보가 분실·도난·유출·위조·변조 또는 훼손되지 않도록 접근 권한 제한, 암호화, 접속기록 관리 등 필요한 보호조치를 적용합니다.
