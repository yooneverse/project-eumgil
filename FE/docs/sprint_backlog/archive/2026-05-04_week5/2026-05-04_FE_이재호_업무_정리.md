## 1. 5주차 책임 범위

이재호의 5주차 책임은 `로그인 -> 분기 -> 홈/검색 -> 길찾기 -> 종료/평가` 핵심 시연 흐름과, 그 흐름을 지탱하는 `마이페이지/회원탈퇴`, `공통 라운딩/섀도우`, `아이콘 인벤토리 문서화`를 마감하는 것이다. 모든 기능에는 QA와 테스트 작업을 별도 Task로 둔다.

| 책임 흐름 | 5주차 종료 기준 |
| --- | --- |
| `AUTH-001` 로그인 3종 실연동 | 카카오, 네이버, 구글 로그인 진입과 서버 인증 응답 분기가 안정적으로 동작 |
| 앱 시작 분기 / 약관 저장 | 일반/보행약자 메인 진입 분기와 약관 저장 API가 실제 흐름에서 정합하게 연결 |
| `MAP-01`, `DEST-*` | 홈/지도/검색 화면이 최신 디자인 기준과 API 응답 기준으로 마감 |
| `ROUTE-01`, `ROUTE-02`, `NAV-01`, `ARR-*` | 길찾기 디자인, 외부링크, 종료/평가 API, 경로저장 연결이 MVP 흐름 기준으로 정리 |
| `MYP-*` | 마이페이지 디자인, 외부링크, 회원탈퇴 API가 시연 가능한 수준으로 정리 |
| 공통 UI / 아이콘 문서 | 버튼/카드 라운딩, 섀도우, 아이콘 인벤토리 문서와 화면/API 매핑이 정리 |
