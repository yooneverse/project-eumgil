## 5. 알려진 환경 이슈 / Skip 사유

| 항목 | 사유 | 후속 |
| --- | --- | --- |
| `gradlew :app:testDebugUnitTest` 실행 실패 | 프로젝트 경로의 한글 문자(`자율 PJT`) + Windows JVM 기본 인코딩(CP949) 충돌 | Android Studio IDE 테스트 러너로 우회 실행. CI 환경(영문 경로)에서는 정상 실행 예상 |
| **LOW_VISION 흐름 fallback 미적용** | `TermsGuideRoute.onRequestDetails` → `context.startActivity(intent)` 직접 호출. `ActivityNotFoundException` 발생 시 크래시 가능 | 별도 보강 Task — T12 패턴(runCatching + Toast) 적용 권장 |
| **마이페이지 흐름 fallback 미적용** | `MyPageAppInfoRoute.kt` `onServiceTermsClick`/`onPrivacyPolicyClick` → `context.startActivity()` 직접 호출. fallback 없음 | 별도 보강 Task — T12 패턴 적용 권장 |
| **URL 정합성** | TermsGuideContract.kt(`?source=copy_link` 포함) vs MyPageAppInfoRoute.kt(쿼리 없음). 같은 페이지지만 문자열 다름 | 별도 보강 Task — 단일 상수 위치(예: `core/config/AppLinks.kt`)로 통합 권장 |
| **Notion 페이지 자체 가용성** | Notion 서비스 장애 시 약관 미노출. 앱 측에서 제어 불가 | 정책 차원 — 정상 운영 가정 |

> Story 4 마감 시점에 발견된 후속 보강 항목은 시연 후 별도 Task로 등록·처리. 시연 회귀에서 위 한계 항목이 막히면 우선 hotfix.
