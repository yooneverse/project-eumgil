| **`reportId` 노출 안 함** | T9 결정사항. 보편적 UX 기준 | 디버깅 필요 시 별도 토글 검토 |
| **제보 이미지 presigned 업로드 환경 확인** | FE는 presigned upload 후 `imageObjectKeys`를 제출한다. 운영 S3/MinIO, CORS, URL 만료 설정이 맞지 않으면 사진 업로드가 실패할 수 있다. | BE/storage 환경 점검 |
| **재시도 시 BE 중복 등록 가능성** | 멱등성 키 미정의. 네트워크 race 시 같은 제보 중복 등록 가능 | BE 협업 — 멱등성 키 도입 별도 Task |
