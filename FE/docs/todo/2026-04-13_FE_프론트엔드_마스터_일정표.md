- **미확정 처리:** BE 연동이 필요한 항목은 API URL, 필드, enum, 인증 상세를 임의 확정하지 않고 `BE 확인 필요`, `추후 구현`, `추가 확인 필요`로 남긴다.

## 7. 완료 기준

- 화면명과 일정 단위는 최신 화면 ID 체계로 표기한다.
- 하단 탭은 `홈/북마크/제보/마이페이지` 4탭 기준으로만 설명한다.
- 북마크 일정은 `북마크`/`BMK` 체계를 사용한다.
- `ROUTE-01`은 도보/대중교통 탭을 모두 포함하고, 경로 옵션은 `안전한 길`, `최단거리` 2개만 둔다.
- BE 연동이 필요한 항목은 `BE 확인 필요`, `추후 구현`, `추가 확인 필요`, `미정` 중 하나로 남긴다.
- 알림, 고객센터/1:1 문의 실제 연결, 길안내 중도 종료 정책, 접근성 필터 더보기 항목, 활동 배지 조건은 MVP 필수 작업과 분리한다.
